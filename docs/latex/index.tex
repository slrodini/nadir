 {\ttfamily nadir} is a minimization library that includes a family of variants of the Adam base algorithm and in the future will include a number of global, non-\/gradient based, minimization algorithm.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{index_autotoc_md1}
{\bfseries{Prerequisite\+:}}
\begin{DoxyEnumerate}
\item You should have installed a C++ compiler with support for C++20.
\item You should have installed {\ttfamily cmake}
\item You should have install the \href{https://eigen.tuxfamily.org/}{\texttt{ Eigen library}}, version \texorpdfstring{$>$}{>}= 3.\+3.
\begin{DoxyEnumerate}
\item Note {\itshape i)} If you install Eigen in a directory not searched by default, you can instruct {\ttfamily cmake} to where to look for Eigen via {\ttfamily -\/DEigen3\+\_\+\+DIR=\texorpdfstring{$<$}{<}Eigen-\/install-\/prefix\texorpdfstring{$>$}{>}/share/eigen3/cmake}.
\item Note {\itshape ii)} If you are working with the \href{https://github.com/MapCollaboration/Oros}{\texttt{ Oros library}}, you can use the Eigen distribution that comes with it. In that case, please use {\ttfamily -\/DEigen3\+\_\+\+DIR=\texorpdfstring{$<$}{<}Oros-\/install-\/prefix\texorpdfstring{$>$}{>}/share/\+Oros/eigen3/cmake} when running {\ttfamily cmake}.
\end{DoxyEnumerate}
\end{DoxyEnumerate}

{\bfseries{Installation\+:}} The usual {\ttfamily cmake} procedure. From within {\ttfamily nadir} directory 
\begin{DoxyCode}{0}
\DoxyCodeLine{mkdir\ build\ \&\&\ cd\ build}
\DoxyCodeLine{cmake\ -\/DCMAKE\_INSTALL\_PREFIX=/your/installation/path/\ ..}
\DoxyCodeLine{make}
\DoxyCodeLine{make\ install}

\end{DoxyCode}
 {\itshape Note\+:} By the default, if no prefix specification is given, {\ttfamily nadir} will be installed in the {\ttfamily /usr/local/} directory. If you want (or need) to use a different path, remember to export the {\ttfamily \texorpdfstring{$<$}{<}install-\/prefix\texorpdfstring{$>$}{>}/lib} folder into the {\ttfamily LD\+\_\+\+LIBRARY\+\_\+\+PATH} and the {\ttfamily \texorpdfstring{$<$}{<}install-\/prefix\texorpdfstring{$>$}{>}/bin} folder int the {\ttfamily PATH}.

After installation, you can run 
\begin{DoxyCode}{0}
\DoxyCodeLine{Nadir-\/config\ -\/-\/help}

\end{DoxyCode}
 to get the list of available flags to be used in your project when using {\ttfamily nadir}.

The library can be uninstalled by running\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{make\ clean}
\DoxyCodeLine{xargs\ rm\ <\ install\_manifest.txt}

\end{DoxyCode}
