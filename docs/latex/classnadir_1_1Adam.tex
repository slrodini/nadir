\doxysubsection{Detailed Description}
Main \doxylink{classnadir_1_1Adam}{Adam} class. 

\doxylink{classnadir_1_1Adam}{Adam} minimizer class. It stores the parameters, references to the Cost function and iteration callbacks, and some internal variables. The main function is {\ttfamily minimize} to perform the minimization.
